# About

 Jyaguchi (蛇口), meaning "faucet" in Japanese, is a Discord bot that turns on and off [Mizore](https://github.com/Kyrielight/Izumi/tree/master/coreV5/mizore) when it's ran in docker mode.

# Commands
```
j.help
j.start
j.stop
j.status
j.ping
```

# Getting Started

1. Fill in these
```toml
[discord]
token = ""
adminID = [""]
```

2. run these commands
```bash
npm i
npm start
```
